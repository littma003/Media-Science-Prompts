# 📹 论文到视频完整工作流

> 从一篇学术论文，到一支成品科普视频的全流程SOP。

---

## 🗺️ 流程总览

```mermaid
graph LR
    A[📄 论文] --> B[🔍 快速解读]
    B --> C[✏️ 脚本生成]
    C --> D[🎨 分镜设计]
    D --> E[🖼️ 画面生成]
    E --> F[🎵 配乐生成]
    F --> G[🎬 视频合成]
```

---

## 阶段1️⃣：论文快速解读

### 提示词 (GPT-4o / Claude)

```markdown
# 角色
你是一位拥有10年经验的科学期刊编辑，擅长将复杂论文转化为通俗解读。

# 任务
请阅读以下论文（或摘要），提取核心信息：

## 输出格式
1. **一句话总结**：用15字以内概括论文的核心发现
2. **研究背景**：这个问题为什么重要？(50字)
3. **核心方法**：研究者怎么做的？(50字)
4. **主要发现**：发现了什么？(3个要点)
5. **现实意义**：这对普通人意味着什么？(30字)
6. **争议/局限**：研究的不足之处(如有)

## 约束
- 禁止添加论文中没有的信息
- 不确定的内容标注 [待核实]
- 使用比喻解释专业术语

# 论文内容
[在此粘贴论文或摘要]
```

## 阶段2️⃣：视频脚本生成

### 提示词

```markdown
# 角色
你是一位B站百万粉科普UP主的金牌编剧，擅长写出让人欲罢不能的解说词。

# 输入信息
{{上一步的论文解读}}

# 任务
基于以上内容，撰写一个3-5分钟的视频脚本。

## 输出格式
| 时间码 | 画面描述 | 解说词 | 情绪节奏 |
|--------|---------|--------|---------|
| 00:00-00:15 | 开场Hook | ... | 悬念/震撼 |
| ... | ... | ... | ... |

## 风格要求
- 前15秒必须有强力Hook（可选：提问式/反常识/数据冲击）
- 每60秒设置一个"信息钩子"防止观众跳出
- 结尾有行动号召(CTA)或思考延伸
- 语言风格：{{专业严谨 / 轻松幽默 / 故事化}}（请选择）

## 约束
- 信息必须与论文一致，不得编造数据
- 使用口语化表达，读起来自然流畅
```

## 阶段3️⃣：分镜设计

### 提示词

```markdown
# 任务
根据以下脚本，生成分镜表。

# 脚本
{{上一步的脚本}}

# 输出格式
| 镜号 | 时长 | 画面类型 | 画面描述 | Midjourney提示词 | 运镜 |
|-----|-----|---------|---------|-----------------|-----|
| 1 | 3s | 实拍参考 | 宇宙深空 | cosmic void, stars... | Zoom in |
| 2 | 5s | 信息图表 | 数据对比 | [使用Canva/手绘] | 静态 |
| ... | ... | ... | ... | ... | ... |

## 画面类型说明
- 实拍参考：用AI图片模拟实拍效果
- 信息图表：数据可视化
- 动画片段：需要动态效果
- 人物出镜：真人或虚拟形象
```

## 💡 Pro Tips

1. **反幻觉策略**：完成脚本后，让AI自己"找茬"
   ```
   请检查上述脚本，标记出所有可能与原论文不符的表述。
   ```

2. **多版本测试**：同一脚本让GPT和Claude各生成一版，取长补短

3. **情绪曲线**：高质量科普视频的情绪应该是波浪式的，而非平直
