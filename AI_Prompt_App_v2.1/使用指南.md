# AI Prompt Library 离线安装包使用指南 (v2.1)

这是一个无需联网、无需服务器环境，在 iPhone 上可以直接运行的高级提示词库 App 压缩包。

## 📁 文件清单
1. `index.html` - 核心应用文件（原 mobile-app-v2.html）
2. `manifest-v2.json` - PWA 配置文件（用于生成桌面图标）
3. `sw.js` - 离线运行和服务更新脚本

## 📲 在 iPhone 上运行并安装的步骤

### 方法 A：通过 OneDrive / 文档应用（最推荐，支持 PWA 安装）
1. 将此文件夹整体上传到您的 **OneDrive** 或 **iCloud Drive**。
2. 在手机上打开 **OneDrive** App。
3. 找到并点击 `index.html`。
4. **关键步骤**：点击分享按钮 -> 选择 **“在浏览器中打开”** (Safari)。
5. 在 Safari 中，点击底部的分享图标 -> 选择 **“添加到主屏幕”**。
6. 完成！现在桌面上会有一个独立的 App。

### 方法 B：本地离线预览
1. 将此文件夹通过 AirDrop 发送到 iPhone。
2. 在 **“文件”** App 中直接打开 `index.html` 即可立即使用所有功能。

---

## 🔄 如何升级提示词？
本应用逻辑与数据全在一个文件中：
1. 当您需要更新提示词时，只需替换新版本的 `index.html` 即可。
2. 即使没有网络，已经加载过的模版也会储存在手机本地缓存中。
