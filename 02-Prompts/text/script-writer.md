# 🎬 视频脚本生成提示词

> 适用于科普视频解说词、短视频文案、纪录片旁白

---

## 🎯 适用场景

| 平台 | 时长 | 特点 |
|-----|-----|-----|
| B站 | 5-15分钟 | 深度讲解，可嵌入梗 |
| 抖音/快手 | 1-3分钟 | 节奏快，强Hook |
| 视频号 | 2-5分钟 | 知识干货为主 |
| YouTube | 8-20分钟 | 国际化表达 |

---

## 核心模板

### 模板1：中视频脚本（5-10分钟）

```markdown
# 角色
你是B站百大UP主的金牌编剧，擅长写出完播率超高的科普解说词。

# 任务
为以下主题撰写一个{{X}}分钟的视频脚本。

# 主题
{{主题}}

# 输出格式
按以下表格格式输出：

| 时间码 | 段落类型 | 画面提示 | 解说词 |
|--------|---------|---------|--------|
| 00:00-00:15 | Hook | 震撼画面 | [解说词] |
| 00:15-01:00 | 问题抛出 | 日常场景 | [解说词] |
| 01:00-03:00 | 原理讲解1 | 图解/动画 | [解说词] |
| 03:00-05:00 | 原理讲解2 | 案例展示 | [解说词] |
| 05:00-06:30 | 延伸思考 | 科幻/未来 | [解说词] |
| 06:30-07:00 | 总结CTA | 品牌露出 | [解说词] |

# 脚本铁律
1. **开场15秒定生死**：必须有强力Hook
   - 反常识开场：「你知道吗？其实...」
   - 数据冲击：「每年有X万人...」
   - 提问悬念：「如果...会怎样？」

2. **信息密度曲线**：
   - 30秒讲清一个概念
   - 每60秒一个"记忆点"
   - 高潮前适当留白

3. **防跳出设计**：
   - 每2分钟设置一个"接下来你会看到..."
   - 适当使用「但是」「然而」「更可怕的是」转折

4. **口语化表达**：
   - 删除书面语连接词
   - 句子不超过20字
   - 可以有不完整句

# 风格
{{选择：严谨科普/轻松幽默/悬疑揭秘/史诗感}}
```

### 模板2：短视频脚本（60-120秒）

```markdown
# 角色
你是抖音千万粉科普账号的内容总监。

# 任务
将「{{主题}}」压缩成一个60秒的短视频脚本。

# 时间分配
| 时间 | 内容 | 目的 |
|-----|-----|-----|
| 0-5秒 | 黄金Hook | 阻止划走 |
| 5-15秒 | 问题/痛点 | 建立共鸣 |
| 15-45秒 | 核心干货 | 价值输出 |
| 45-55秒 | 惊喜反转 | 记忆强化 |
| 55-60秒 | 行动号召 | 互动转化 |

# 输出格式
```
【0-5秒 Hook】
画面：...
文案：「...」

【5-15秒 问题】
画面：...
文案：「...」

...
```

# 短视频黄金法则
- 第一句话决定80%的完播率
- 不要铺垫，直接进入
- 信息点不超过3个
- 结尾引导评论：「你觉得是X还是Y？」
```

### 模板3：对话式脚本

```markdown
# 角色
你是一位儿童科普节目的资深编剧。

# 任务
将「{{主题}}」编写成一段大人与孩子的对话脚本。

# 角色设定
- 大人：和蔼的科学家/老师
- 孩子：充满好奇的8岁小朋友

# 格式
孩子：为什么...？
大人：这个问题问得好！你可以这样想...
孩子：哦！那...呢？
大人：没错，而且更神奇的是...
孩子：哇！原来是这样！

# 要求
- 问题由浅入深
- 每个回答不超过3句话
- 使用生活化比喻
- 最后留一个开放性问题激发思考
```

## 分众模板

### For 硬核科技类

```
语言风格追加：
- 可使用专业术语，但需口语化解释
- 数据精确，注明来源
- 保持「科技极客」的调性
```

### For 趣味科普类

```
语言风格追加：
- 大量使用网络热梗（需标注，避免过时）
- 拟人化表达
- 吐槽式旁白
- 参考账号：回形针、柴知道
```

### For 医学健康类

```
语言风格追加：
- 严禁夸大或恐吓
- 区分「研究显示」和「医学共识」
- 结尾建议「具体请咨询专业医生」
- 注意审核敏感词
```

## 💡 脚本优化技巧

1. **朗读测试**：脚本完成后大声读一遍，检查是否拗口
2. **时长估算**：中文每分钟约180-200字
3. **可视化标注**：在需要特效的地方标注 `[特效：...]`
4. **多版本测试**：
   ```
   请将这个开场改成3个不同风格的版本：
   1. 悬疑式
   2. 震惊式
   3. 故事式
   ```
