# 🎯 标题优化提示词

> 标题决定80%的点击率，本文档专注于科普内容标题的打磨。

---

## 📐 标题公式大全

### 公式1：数字+利益

```
格式：[数字] + [对象] + [利益/结果]
例子：
- 「5个习惯，让你的大脑年轻10岁」
- 「99%的人都不知道的3个物理冷知识」
- 「每天10分钟，轻松理解量子力学」
```

### 公式2：悬念反转

```
格式：[表象] + [转折] + [真相]
例子：
- 「你以为是缺钙？其实是...」
- 「科学家说了20年的话，居然是错的」
- 「看似正确的健康建议，正在伤害你」
```

### 公式3：权威背书

```
格式：[权威来源] + [发现/结论]
例子：
- 「哈佛研究：睡前做这件事，记忆力提升40%」
- 「Nature 最新论文揭示：...」
- 「NASA工程师推荐的...」
```

### 公式4：恐惧驱动

```
格式：[日常行为] + [潜在风险]
例子：
- 「你每天都在用的东西，可能正在损害...」
- 「90%的人都有这个坏习惯，后果很严重」
- 「千万别再...了，专家警告...」
```

### 公式5：好奇缺口

```
格式：[问题] + [暗示有答案]
例子：
- 「为什么飞机窗是圆的？答案你想不到」
- 「月球背面到底有什么？真相终于揭开」
- 「这个问题难倒了爱因斯坦，你能想明白吗？」
```

### 公式6：对比冲击

```
格式：[A] vs [B] + [结果]
例子：
- 「中国 vs 美国科技实力，真实差距多大？」
- 「10年前 vs 10年后，科学如何改变世界」
- 「传统方法 vs AI生成，结果令人意外」
```

## 标题优化追问

### 变得更吸引人

```
这个标题不够吸引人，请用以下角度优化：
1. 增加紧迫感
2. 加入反常识元素
3. 使用更口语化的表达
```

### 降低标题党感觉

```
这个标题太像标题党了，请保持吸引力的同时让它更真诚、更有信息量。
```

### 适配多平台

```
请将这个标题分别适配：
1. B站视频标题（25字以内）
2. 抖音标题（15字以内）
3. 公众号头条（20字以内）
4. 小红书笔记（emoji+关键词）
```

## 标题检查清单

```markdown
# 标题自检清单

## 吸引力检验
- [ ] 能在3秒内看懂核心信息
- [ ] 有明确的利益点或好奇点
- [ ] 目标读者看了会想点进去

## 真实性检验
- [ ] 标题与内容相符，不是标题党
- [ ] 没有过度承诺或夸大
- [ ] 数据/结论来源可靠

## 平台适配检验
- [ ] 长度符合平台限制
- [ ] 关键词靠前放置
- [ ] 没有违禁词/敏感词

## 可读性检验
- [ ] 读起来顺口
- [ ] 没有歧义
- [ ] 标点使用正确
```

## 💡 Title 迭代工作流

```
原始标题 → AI生成10个变体 → 筛选前3 → 人工优化 → A/B测试 → 确定最终
```

示例流程：

```markdown
1. 原始想法：讲讲黑洞

2. AI扩展：
   - 「如果你掉进黑洞，会发生什么？」
   - 「黑洞的5个你不知道的事实」
   - 「这个黑洞，正在吞噬银河系」

3. 人工优化：
   - 「黑洞真的会把你压成面条吗？物理学家告诉你真相」

4. 最终版本：
   - 「如果你掉进黑洞...（物理学家模拟了全过程）」
```
