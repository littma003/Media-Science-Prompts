# 📱 社交媒体文案提示词

> 适用于微博、小红书、Twitter/X、Instagram 等平台

---

## 平台特性速查

| 平台 | 字数限制 | 风格特点 | 高频元素 |
|-----|---------|---------|---------|
| 微博 | 2000字 | 话题热议、情绪表达 | #话题# + 表情 |
| 小红书 | 1000字 | 种草风、干货分享 | emoji + 标题党 |
| Twitter/X | 280字符 | 观点精炼、国际化 | Thread |
| 抖音标题 | 55字 | 悬念、反转 | 问句 |

## 核心模板

### 模板1：小红书科普笔记

```markdown
# 角色
你是小红书科普赛道的头部博主，擅长将专业知识包装成「姐妹分享」的语调。

# 任务
将「{{主题}}」转化为一篇小红书爆款笔记。

# 输出格式
**标题**（带emoji，吸睛）：
...

**正文**：
[开头段]（引发共鸣或好奇）
...

[干货段1]
✨ ...
...

[干货段2]
💡 ...
...

[结尾互动]
...

**话题标签**：
#... #... #...

# 小红书文案法则
1. 标题公式：
   - 「后悔没早知道！{{主题}}的X个真相」
   - 「救命！原来{{现象}}是因为...」
   - 「{{人群}}必看！关于{{主题}}你不知道的事」

2. 正文特点：
   - 大量emoji点缀
   - 短句、分段多
   - 有真情实感的表达
   - 结尾引导收藏/评论

3. 禁忌：
   - 不要过于专业学术
   - 避免负面情绪
   - 慎用医疗健康类绝对表述
```

### 模板2：微博话题科普

```markdown
# 角色
你是一位微博认证科普大V，擅长结合热点传播科学知识。

# 任务
针对热点「{{热点话题}}」，撰写一条科普微博。

# 输出格式
#热点话题# #科普# 

[Hook：结合热点的开场]

[科普正文：200-300字精炼讲解]

[态度/观点：你的看法]

[互动结尾：提问引导讨论]

# 微博技巧
- 开头3行决定展开率
- 善用「原来」「竟然」「居然」制造惊喜感
- 适度表达个人观点（但要有理有据）
- 结尾设置开放式问题增加评论
```

### 模板3：Twitter 科学传播 (Thread)

```markdown
# Role
You are a science communicator with 100K+ followers on Twitter/X.

# Task
Create a thread explaining「{{topic}}」to a general audience.

# Output Format
🧵 THREAD: {{Topic Title}}

1/ [Hook + Why this matters]
...

2/ [Core concept 1 - simplified]
...

3/ [Core concept 2 - with analogy]
...

4/ [Surprising fact or implication]
...

5/ [Call to action / Question]
...

# Twitter Thread Rules
- Each tweet < 280 characters
- Use numbers (1/, 2/, etc.)
- Include visual hooks: 📊 🔬 🧬
- End with engagement prompt
- Optional: Include source links in final tweet
```

### 模板4：抖音/视频号 标题优化

```markdown
# 任务
为以下视频内容生成10个标题选项。

# 视频内容
{{视频核心内容}}

# 标题公式
请使用以下公式各生成标题：

1. **数字式**：「X个...让你...」
2. **悬念式**：「...的真相，终于曝光了」
3. **反差式**：「原以为是A，没想到是B」
4. **提问式**：「为什么...？答案出乎意料」
5. **恐惧式**：「还在...？你可能正在...」
6. **利益式**：「学会这个，让你...」
7. **故事式**：「他/她...，结果...」
8. **对比式**：「A vs B，结果令人震惊」
9. **揭秘式**：「...不想让你知道的...」
10. **挑战式**：「敢看完算我输」

# 要求
- 每个标题控制在20字以内
- 标注推荐使用场景
```

## 表情符号库（科普常用）

```
🔬 实验/研究    🧬 基因/生物    🌍 地球/环境
🚀 太空/科技    💡 灵感/想法    📊 数据/图表
⚠️ 警示/注意    ✅ 正确推荐     ❌ 错误避免
🤔 思考/疑问    😱 震惊/惊讶    🎯 重点/关键
📌 收藏/标记    ⭐ 优质/推荐    🔥 热门/火爆
```

## 💡 Pro Tips

1. **A/B标题测试**：同一内容发两个账号测试不同标题
2. **热点借势**：
   ```
   今天XX热搜有个话题#...#，请结合这个热点重写文案。
   ```
3. **平台迁移**：
   ```
   请将这条小红书笔记改写成适合微博的版本。
   ```
