## 💡 中文Prompt技巧

### 1. 使用四字成语描述情绪

```
气势磅礴、波澜壮阔、行云流水、
娓娓道来、引人入胜、豁然开朗
```

### 2. 引用具体场景

```
"像《舌尖上的中国》那种感觉"
"类似《国家宝藏》的开场"
"央视纪录片风格"
```

### 3. 明确指定乐器

```
主奏：钢琴/小提琴/二胡/竹笛
伴奏：吉他/轻柔弦乐/电子合成器
节奏：架子鼓/轻拍手/无打击乐
```
